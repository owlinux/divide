Window Resize Using Keyboard Shotcuts

Window Resize and Move just Easy

No Mouse and No Trackpad, Only Shortcuts Keyboard

Window Action

Windows can be moved to a number of predefined regions of the scree

    • Full Screen : ⌥⌘F
    • Left Half Screen : ⌥⌘←
    • Right Half Screen : ⌥⌘→
    • Top Half Screen : ⌥⌘↑
    • Bottom Half Screen : ⌥⌘↓
    • Upper Left Top : ⌃⌘←
    • Lower Left Bottom: ⌃⇧⌘←
    • Upper Right Top : ⌃⌘→
    • Lower Right Bottom : ⌃⇧⌘→
